# nestjs-demo

## Description

[Nest](https://github.com/nestjs/nest) + MySQL + Swagger + JWT Authorization starter repository. 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# hot reload
$ npm run start:hrm

# production mode
$ npm run start:prod
```
